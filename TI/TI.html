Técnico em Informática

Aqui, você verá um pouco do que é apresentado aos alunos do E.M ao longo dos 3 anos de curso na área de Técnico em informática (T.I).

Em T.I vimos lógica de programação que serve para construir programas no geral, e para isso usamos programação em Blocos no programa Scratch, o que nos ajudou a entender a lógica. A programação em Javascript é o tipo de linguagem de programação usada em aplicações gerais como o iFood por exemplo. Também vimos Banco de Dados que serve para armazenar os dados e informações de uma empresa ou usuário, para fazer isso usamos a linguagem de programação SQL. Estamos aprendendo recentemente HTML para criar um site.
